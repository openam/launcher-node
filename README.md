## Install Steps
  * [Install node.js](https://github.com/joyent/node/wiki/Installation)

```sh
    $ git clone git://github.com/openam/launcher-node.git
    $ cd launcher-node
    $ npm install
    $ node app.js
```

  * [http://localhost:3001/](http://localhost:3001/)
